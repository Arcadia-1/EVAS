EVAS — 事件驱动 Verilog-A 仿真器
====================================

EVAS 是一个轻量级、纯 Python 实现的数字/混合信号 Verilog-A 行为仿真器，
提供快速的事件驱动仿真，无需任何外部 EDA 工具依赖。
